AI Destekli Borsa ve Finans Haber Botu (Content-Bot)

Bu proje, Python kullanarak finans haberlerini otomatik olarak toplayan, Yapay Zeka (Google Gemini) ile analiz eden ve sonuçları HTML bülten formatında e-posta(Gmail) ile raporlayan tam kapsamlı bir otomasyon sistemidir.


-------------

 Scraping: Selenium ile Bloomberg HT, Ekonomim gibi sitelerden dinamik veri çekimi.
 Veri Saklama: SQLite veritabanı ile verilerin yapılandırılmış şekilde saklanması ve dublikasyon kontrolü.
 Yapay Zeka Analizi: Google Gemini API kullanılarak haberlerin özetlenmesi ve duygu analizi 
 Raporlama: Jinja2 şablon motoru ile dinamik ve şık HTML bülten tasarımı.
 Otomasyon:SMTP protokolü ile kişiselleştirilmiş e-posta gönderimi.


--------------------------
 
 
 Kullanılan Teknolojiler

#Python 3.10+, #Google Gemini Pro API, #Selenium WebDriver,#SQLite, #Jinja2, #Poetry&Dotenv
